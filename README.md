# Attendance Crisis Calculator 📊💀

> *Bunk responsibly... ya phir mat karo 😅*

### Kya hai ye? 
Ek 1-min web tool jo batata hai ki 75% attendance maintain karne ke liye tum **kitni aur classes bunk kar sakte ho**. 

Because "proxy laga dunga" is not a career plan 😂

### ✨ Features
- 🔢 **Instant Calculation** - Total classes aur attended daalo, % turant pata chalo
- 😰 **Crisis Mode** - 75% se kam hue toh red alert + "Parents Call Incoming" warning
- 😎 **Bunk Counter** - Exact batayega "3 bunk aur maar sakte ho" ya "Beta padhai kar le"
- 📱 **Mobile Friendly** - Class me last bench pe baith ke bhi check kar sakte ho
- 🌙 **No Login Needed** - Bas kholo aur use karo. Privacy 100%

### 🛠️ Built With
- HTML + CSS + Vanilla JS - Because simplicity is best
- 0 Libraries - Loads faster than your prof's attendance sheet
- 15 Minutes of Code + 3 Years of College Trauma 😭
